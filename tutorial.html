To install Sigma 5.0, simply drag and drop all the files from the ".minecraft.zip" that you downloaded into %appdata%/.minecraft and click on "replace files in destination" if Windows asks you to replace them. Make sure you drag and drop everything.

To install Sigma 4.11 / 3.9, simply extract the "Sigma" folder from the zip file into ".minecraft/versions" like you would normally do with any regular client. Back to Home
